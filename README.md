# Chrono Prestige

Chrono Prestige est une boutique en ligne de montres de luxe développée dans le cadre d'un projet universitaire en L1, réalisée en groupe de 4 personnes.

## Fonctionnalités

- Achat et vente de montres de luxe
- Panier d'achat
- Liste de favoris
- Inscription et connexion
- Espace vendeur
- Espace administrateur

## Technologies utilisées

- PHP, MySQL
- HTML, CSS
